# Тест задание


Необходимо на PHP или Node.js, Vuejs написать код который с помощью вебхука обратится к Битрикс24 по рест апи и заберёт список из 10000 компаний. 

Результат ( список Компаний) вывести любым удобным образом.

Код разместить в гите.

# Результат

Код находится в `./src/App.vue`

Подстановка хуков: 

в строке 26 нужно заменить 'https://your-domain.bitrix24.ru/rest/crm.company.list.json' на адрес bitrix,

в строке 28 нужно заменить 'your-auth-token-here' на bitrix - токен.

Запуск приложения: 

```
npm install
npm run dev
```
